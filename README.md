## Installing Necessary Dependencies

```
chmod +x run_both.sh

./run_both.sh
```

## Running "run_both" file for automatic utilization of both files (.py and .js)
### For MacOS/Linux
```
pip install selenium
pip install beautifulsoup4
npm install puppeteer
```
### For Windows
```
run_both.bat
```
